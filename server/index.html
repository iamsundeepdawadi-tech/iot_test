Documentation

1. Import Database
## cPanel → phpMyAdmin → Import → schema.sql ## Or via SSH: mysql -u aitalimc_admin -p aitalimc_iot < schema.sql
2. Add a Device

Go to Devices → + Add New. Pick a device type, fill details, click Register. Then click View Code on the device card to get Arduino IDE code.

3. Arduino IDE Setup
Board URL (Settings → Additional Boards): https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json Install Libraries (Tools → Manage Libraries): - "DHT sensor library" by Adafruit - "ArduinoJson" by Benoit Blanchon Board: "ESP32 Dev Module" Upload Speed: 115200
4. Flash & Test

Paste generated code into Arduino IDE → Upload. Open Serial Monitor (115200 baud). ESP32 will connect to WiFi and start sending data. Dashboard shows data within seconds.

ISR Best Practices

• Use ARDUINO_ISR_ATTR for ISRs to run from IRAM
• Mark shared vars as volatile
• Use millis() not delay() for non-blocking timing
• Keep ISRs short — set flags only, handle in loop()
